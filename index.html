Unsung Women

Influential Woman Wikipedia-inspired Library

"Unsung Women" is a crowd-sourced platform that brings to light the contributions of historically significant women whose accomplishments are not widely recognized. This project allows users to contribute and edit content in a collaborative manner, ensuring these women receive the recognition they deserve in history.

As a full-stack developer, I implemented the front-end with React for dynamic user interfaces and Tailwind CSS for custom, responsive layouts. The back-end was built using Flask, leveraging PostgreSQL for database management, with SQLAlchemy for ORM capabilities. Secure authentication is handled through PyJWT, with Flask-Bcrypt for password hashing.

Pacman GA Edition

Welcome to the Pacman GA Edition, a modern twist on the classic Pacman game.

In this edition, players embark on a nostalgic journey with a unique GA-themed twist. As always, the objective is for Pacman to devour all the pellets scattered across the board while evading the four iconic ghosts: Blinky, Pinky, Inky, and Clyde.

Failure to do so risks losing one of Pacman's three precious lives. Crafted with HTML, CSS, and JavaScript, this game promises an engaging experience as players navigate through an original, GA-themed grid.
